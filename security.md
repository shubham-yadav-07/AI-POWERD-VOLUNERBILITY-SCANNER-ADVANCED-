# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 3.0     | ✅ Yes    |
| < 3.0   | ❌ No     |

## Ethical Use Only

This tool must only be used on:
- Systems you own
- Systems you have written permission to test
- Lab/test environments (e.g. DVWA, HackTheBox, TryHackMe)

## Reporting a Vulnerability

If you find a security issue in this project itself, please open a GitHub Issue marked **[SECURITY]** rather than disclosing publicly.
